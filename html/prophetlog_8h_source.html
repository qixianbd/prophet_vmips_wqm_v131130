00028 /* Logging Facility
00029         Prophet logs are divided into four levels:
00030         pdebug  :       debug messages that are normally suppressed
00031         pinfo   :       informationally messages that are normally shown
00032         pwarn   :       warning messages that signal a problem
00033         perror  :       error messages that signal major and unvoeralble problems
00034 
00035         perror will automatically crash the process after it show the error message.
00036 
00037         usage: If you want to show some log message in your class member function
00038                 (normal or static), just use the PLOG_CLASS macro, this will cause 
00039                 you log message tagged with you class name and function name.
00040                 example:
00041                 class Foo
00042                 {
00043                         PLOG_CLASS(Foo);
00044                 public:
00045                         void show()
00046                         {
00047                                 pinfo<<"Foo class"<<pendl;
00048                         }
00049                 }
00050                 the output will be:
00051                 PROPHET_INFO: Foo::show(): Foo class
00052                 NOTE: pendl MUST be called at the end of the log printing statement.
00053                         This version is not multithread safe.
00054 */
